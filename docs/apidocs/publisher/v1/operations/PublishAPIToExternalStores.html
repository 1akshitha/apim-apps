Publish API To External Stores

Publish an API to external stores

POST https://apis.wso2.com/api/am/publisher/v1.0/apis/{apiId}/publish-to-external-stores

This operation can be used to publish an API to a list of external stores.

OAuth 2.0 Scope

apim:api_publish
